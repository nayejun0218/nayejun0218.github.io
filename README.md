# The Bridge Community Church Website

더브릿지커뮤니티 교회 공식 홈페이지 프로젝트입니다.

## 소개
- 이 웹사이트는 더브릿지커뮤니티 교회의 소개, 비전, 예배 안내, 사역, 연락처 등 다양한 정보를 제공합니다.
- 심플하고 따뜻한 카드형 디자인으로 구성되어 있습니다.

## 주요 페이지
- 홈 (index.html)
- 교회소개 (about.html)
- 담임목사 소개 (pastor.html)
- 교회 비전 (vision.html)
- 예배시간 안내 (worship.html)
- 찾아오시는 길 (location.html)
- 사역 (ministries.html)
- 연락처 (contact.html)

## 실행 방법
1. 이 폴더에서 터미널을 열고 아래 명령어를 입력하세요:
   ```
   python3 -m http.server 8000
   ```
2. 브라우저에서 [http://localhost:8000](http://localhost:8000) 접속
